# ToolMuse

ToolMuse is a lightweight hub for creative digital tools.

It helps users discover, launch, and work with independent applications designed for creation, experimentation, and productivity.

---

## What’s inside

This repository contains:

- **ToolMuse** — the main interface for navigating tools
- **Event Editor** — an example standalone tool

Each tool runs independently and can evolve on its own.

---

## Vision

ToolMuse is built around simplicity:

- clear navigation
- minimal friction
- tools that do one thing well

More tools and features will be added over time.
